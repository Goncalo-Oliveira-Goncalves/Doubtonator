# Note Popup

The note popup window is very simple in nature, it displays a markdown of what the note's contents are... however it comes with some handy features.

Similar to a feature for answering questions, it has a voice feature -- the button will be the same. And the text will be automatically pasted and formatted by AI while keeping the tone.
